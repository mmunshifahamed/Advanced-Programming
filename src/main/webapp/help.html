Help / User Guide

How to Use the System

  1. Login using your username and password.
  2. Add Reservation from the dashboard and fill all fields correctly.
  3. View Reservation by entering a reservation number to see details.
  4. Calculate Bill by entering the reservation number (bill shows Rs. values).
  5. View All Reservations to see a report of all reservations.
  6. Logout when you finish.

Common Errors

  • Reservation number already exists: use a new unique reservation number.
  • Contact must be 10 digits: enter only numbers (e.g., 0771234567).
  • Check-out must be after check-in: correct the dates.
  • Please login first: login again if session expired.
