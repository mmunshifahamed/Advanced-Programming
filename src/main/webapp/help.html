Quick guide to use the Hotel Reservation System

⬅ Dashboard

✅ How to Use the System

Follow these steps to complete tasks smoothly.

  1. Login using your username and password.
  2. Add Reservation from the dashboard and fill all fields.
  3. View Reservation using a reservation number to see details.
  4. Calculate Bill using the reservation number (bill shows Rs. values).
  5. View All Reservations to see a report of all reservations.
  6. Logout when you finish.

⚠️ Common Errors

If you see these messages, here’s what to do.

  • Reservation number already exists → Use a new unique number.
  • Contact must be 10 digits → Use only numbers (e.g., 0771234567).
  • Check-out must be after check-in → Fix the date selection.
  • Please login first → Session expired, login again.

💡 Tips

Small tips to avoid mistakes.

  • Keep reservation numbers like R001, R002 (easy to track).
  • Use Sri Lankan phone format: 07XXXXXXXX.
  • Check-in and check-out must be valid dates (out date should be later).
  • Use View All Reservations to confirm saved data.
