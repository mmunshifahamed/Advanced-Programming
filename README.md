Room Reservation System

The Room Reservation System is a web-based application developed to manage hotel room bookings efficiently using Java-based technologies. The system was built using Java Servlets for the backend, HTML and CSS for the user interface, and MySQL as the database. Its main objective is to replace manual reservation processes with a computerized system that improves booking management, reduces human errors, and organizes reservation data effectively. By using this system, hotel staff can easily handle reservations, view booking details, and manage guest information through a simple and secure interface.

The system provides several key functionalities to support reservation management. It includes a secure login system that allows only authorized staff members to access the application. After logging in, users can add new reservations by entering guest details such as name, contact information, room type, and check-in/check-out dates. The system also allows users to view all reservations, search reservations using a reservation number, update or delete bookings, and generate bills automatically based on the room type and number of nights stayed. Input validation mechanisms are included to prevent incorrect entries such as invalid dates or missing information.

The application follows a three-tier architecture consisting of the presentation layer, application layer, and data layer. The presentation layer includes the HTML and CSS pages that interact with users through a web browser, while the application layer uses Java Servlets to process requests and implement system logic. The data layer consists of a MySQL database that stores user accounts and reservation records. Version control for the project was managed using Git and GitHub, which helped track changes, maintain different versions of the system, and ensure secure backup of the project files. Overall, the system demonstrates how web technologies can be used to automate hotel reservation management and improve operational efficiency
